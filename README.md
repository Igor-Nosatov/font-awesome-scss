#flexible-fa
Flexible Font Awesome by @dnomak Â· http://dnomak.com

## Using Sass
```html
<ul>
  <li><a href="#">Github</a></li>
  <li><a href="#">Twitter</a></li>
</ul>
```
####1. Import
```scss
@import "flexible-fa";
```
####2. Icon
```scss
%fa {
  @include fa();
}
ul li:fist-child a:before {
  @extend fa();
  @include fa(github);
}
ul li:last-child a:before {
  @extend fa();
  @include fa(twitter);
}
```
#### 3. Size
```scss
%fa {
  @include fa();
}
ul li:fist-child a:before {
  @extend fa();
  @include fa(github);
  @include fa-size(2em);
}
ul li:last-child a:before {
  @extend fa();
  @include fa(twitter);
  @include fa-size(4em);
}
```
####4. Align
```scss
// top, middle, bottom
%fa {
  @include fa();
}
ul li:fist-child a:before {
  @extend fa();
  @include fa(github);
  @include fa-align(top);
}
ul li:last-child a:before {
  @extend fa();
  @include fa(twitter);
  @include fa-align(bottom);
}
```
#### 5. Rotated
```scss
%fa {
  @include fa();
}
ul li:fist-child a:before {
  @extend fa();
  @include fa(github);
  @include fa-rotate(90deg);
}
ul li:last-child a:before {
  @extend fa();
  @include fa-rotate(180deg);
}
```
####6. Flipped
```scss
// horizontal, vertical
%fa {
  @include fa();
}
ul li:fist-child a:before {
  @extend fa();
  @include fa(github);
  @include fa-flip(horizontal);
}
ul li:last-child a:before {
  @extend fa();
  @include fa-flip(vertical);
}
```
####7. Spin
```scss
%fa {
  @include fa();
}
ul li:fist-child a:before {
  @extend fa();
  @include fa(github);
  @include fa-spin(2s);
}
ul li:last-child a:before {
  @extend fa();
  @include fa-spin(4s);
}
```

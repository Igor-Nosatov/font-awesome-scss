#Font Awesome Plus
Font Awesome Plus by @dnomak · http://dnomak.com

## Using Sass

####Import
```scss
@import "font-awesome-plus";
```

####Html
```html
<ul>
  <li><a href="#">Github</a></li>
  <li><a href="#">Twitter</a></li>
</ul>
```

####1. Icon
CodePen · http://codepen.io/dnomak/pen/EaWKmr
```scss
%fa {
  @include fa();
}
ul li:first-child a:before {
  @extend %fa;
  @include fa(github);
}
ul li:last-child a:before {
  @extend %fa;
  @include fa(twitter);
}
```

#### 2. Size

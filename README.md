#Flexible Font Awesome
Flexible Font Awesome Unofficial by dnomak https://github.com/dnomak

## Using

**Html**
```html
<ul>
  <li><a href="#">Github</a></li>
  <li><a href="#">Twitter</a></li>
</ul>
```
--
###1. Import
**Sass**
```scss
@import "flexible-fa";
```

**Less**
```less
@import "flexible-fa.less";
```

**Stylus**
```
@import "flexible-fa"
```

--
###2. Icon

**Sass**
```scss
%fa {
  @include fa();
}
ul li:fist-child a:before {
  @extend fa();
  @include fa(github);
}
ul li:last-child a:before {
  @extend fa();
  @include fa(twitter);
}
```

**Less**
```less
.fa {
  fa();
}
ul li:fist-child a:before {
  &:extend(.fa);
  .fa(github);
}
ul li:last-child a:before {
  &:extend(.fa);
  .fa(twitter);
}
```

**Stylus**
```
.fa
  fa()
ul li:fist-child a:before
  @extend .fa
  fa(github)
ul li:last-child a:before
  @extend .fa
  fa(twitter)
```

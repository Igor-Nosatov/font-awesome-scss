# font-awesome-scss
Font Awesome Scss Plus

## Install

```scss
@import "font-awesome";
```

## Using

### Html
```html
<ul>
  <li><a href="#">Github</a></li>
  <li><a href="#">Twitter</a></li>
</ul>
```

### Icon
```scss
ul li:first-child a:before {
  @include fa;
  @include fa(github);
}
ul li:last-child a:before {
  @include fa;
  @include fa(twitter);
}
```

### Size
```scss
ul li:first-child a:before {
  @include fa;
```
